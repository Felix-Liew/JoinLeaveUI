# JoinLeaveUI

A modern Join and Leave UI system that shows animated in-game notifications when players enter or exit the server, creating a more lively and engaging server experience.

Enhance your server atmosphere with smooth transitions, clean visuals, and customizable messages for each player event.
Fully optimized for performance, ensuring zero lag while keeping your UI visually appealing.
Easy to configure and integrate into any server setup, making it perfect for both small communities and large multiplayer servers.
